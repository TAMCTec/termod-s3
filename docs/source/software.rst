Software
=============

Sense Node C3 is a open platform with ESP32 C3, so it is compatible with ESP-IDF, Arduino IDE, Platform IO, Micropython, CircuitPython, etc. It is your own choice of which platform you are comfortable with. But the most recommecded way to use this, is Home Assistant with ESPHome.
